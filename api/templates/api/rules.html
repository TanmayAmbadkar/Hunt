Rules

  1. This event is a single player event, participants are not allowed to team up. If found, will be disqualified immediately
  2. Every level has a score, which decays by 3% every hour. For eg, if it was 100, then it will be 97 the next hour.
  3. The event lasts for 30 hours for the first year, and 24 hours for the 2nd and 3rd year.
  4. Teaming-up will only help one person win, there are no ties in this game.
  5. The players highest score at the end will be rewarded, as mentioned in the emails
  6. Seniors should not participate in the event for first year unless they wish to face public humiliation. We have records for everything.
  7. Once the contest ends, you may continue to participate, but the leaderboard will stay frozen.
  8. You are solely responsible for network issues from your end. In case we have server issues, we will take necessary steps to ensure you get ample time for the event.
  9. Join the discord server for hints, giveaways and more.
